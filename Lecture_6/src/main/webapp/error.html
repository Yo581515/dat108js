Hendelse error

Hendelsen error fyrer på window ved f.eks. feil i JavaScript-kode eller nettforbindelse. I dette eksempelet kjøres en metode ved error. Denne metoden kansellerer så error hendelsen sin standardoppførsel, som er å vise feilen i nettleser-konsollet. Det gjøres ved å kalle hendelsen sin metoden preventDefault.

All utskrift fra JavaScript-koden skrives til web konsollet.
