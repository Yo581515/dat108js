Hendelser change og input

Hendelsen change og input trigger begge ved endringer i FORM elementer. I utgangspunktet vil hendelse input fyre ved hver endring, mens change når endringen er ferdig, typisk når bruker forlater elementet.

For elementer som SELECT og INPUT av type DATE oppfører change og input seg likt.

For INPUT av type DATE fyrer hendelsene kun hvis endringen involverer en gyldig dato. Ingen hendelser fyrer hvis bruker sletter en ufullstendig dato, eller endrer en ufullstendig dato til en annen ufullstendig dato. For dette elementet kan hendelsen blur være nyttig. Den fyrer når elementet mister fokus.

All utskrift fra JavaScript-koden skrives til web konsollet.
